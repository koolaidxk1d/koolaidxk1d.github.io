Make WSL2 accessible via LAN

First, add a netsh interface.

Powershell:

netsh interface portproxy add v4tov4 `
listenport=X `
listenaddress=0.0.0.0 `
connectport=Y connectaddress=z.z.z.z

Replace X with your desired LAN facing port, Y with your WSL2-side port, and z.z.z.z with the ip of your WSL2 installation (find using hostname -I)

To Undo:

netsh interface portproxy delete v4tov4 `
listenport=X `
listenaddress=0.0.0.0

After adding the interface, you must also create an Inbound Rule on Windows Firewall to allow the port over TCP.
